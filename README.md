# Aplicação desenvolvida em React para gerenciamento de usuários

## Funcionalidades

- Listar usuários
- Adicionar usuários
- Editar usuários
- Remover usuários
- Atualização dinâmica da interface

## Conceitos aplicados

- Componentização
- Props e State
- Eventos no React
- Renderização condicional
- Manipulação de listas

## Tecnologias

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite
